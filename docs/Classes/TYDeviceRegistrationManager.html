  • queryRegistrationResult(of:completionHandle:)

    Runs the polling task to check the device pairing result.

    Device pairing is an asynchronous process. The time when a device is paired depends on the network quality. You must make the API request to check the device pairing result. The polling task is stopped when a list of paired devices is returned or when the pairing task timed out.

    The polling task is run once per second. The timeout value of the pairing task is recommended to be 100 seconds.
